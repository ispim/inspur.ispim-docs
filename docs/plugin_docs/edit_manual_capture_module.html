inspur.ispim.edit_manual_capture – Set manual capture screen.¶

Note

This plugin is part of the inspur.ispim collection.

To install it use: ansible-galaxy collection install inspur.ispim.

To use it in a playbook, specify: inspur.ispim.edit_manual_capture.

New in version 1.0.0: of inspur.ispim

Synopsis¶

  • Set manual capture screen on Inspur server.

Parameters¶

Examples¶

- name: Screen test
  hosts: ism
  connection: local
  gather_facts: no
  vars:
    ism:
      host: "{{ ansible_ssh_host }}"
      username: "{{ username }}"
      password: "{{ password }}"

  tasks:

  - name: "Set manual capture screen"
    inspur.ispim.edit_manual_capture:
      type: "capture"
      provider: "{{ ism }}"
